# Greenhouse Gas Emissions by Sector Dashboard

This project is an interactive Streamlit dashboard developed for the 5DATA004C Data Science Project Lifecycle individual coursework.

## Dataset

The dataset used is the European Environment Agency greenhouse gas emissions dataset. It contains emissions data by country, year, sector, pollutant and unit.

## Features

- Country filter
- Year range filter
- Sector filter
- Emissions trend chart
- Sector comparison chart
- Country comparison chart
- Sector contribution pie chart
- KPI summary cards
- Data preview table

## Tools Used

- Python
- Streamlit
- Pandas
- Plotly
- GitHub
- Google Colab

```bash
pip install -r requirements.txt
